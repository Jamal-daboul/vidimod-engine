🔤  WILLIAM — CUSTOM FONTS FOLDER
==================================

Drop font files in THIS folder and they'll appear in the "Font" picker on the
Create Video page, alongside the built-in fonts.

• Supported: .ttf, .otf, .ttc
• Great free sources: Google Fonts (fonts.google.com), DaFont, Fontsource.
• For ENGLISH viral captions try: Montserrat, Bebas Neue, Anton, Poppins, Inter.
• For ARABIC try: Cairo, Tajawal, Almarai, Changa (download the .ttf).
• After adding files, click the "↻" next to the Font picker (or reload the page).

Note: a font only renders the languages it supports. An English-only display
font will fall back to a default for Arabic text — pick an Arabic font for
Arabic videos.
